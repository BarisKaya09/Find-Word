# Find-Word
Text dosyasındaki kelimeleri bulup çıktı veren command line uygulama.

```go
go run main.go fw "ve" desk\a.txt

// write log file (bug)
go run main.go fw -l "ve" desk\a.txt

// help command
go run main.go fw -help
```
